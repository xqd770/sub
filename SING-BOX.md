# Sing-box 订阅说明

## 使用方法

### 1. 获取订阅链接
- 在转换后的链接前添加 `/api/sing?url=`
- 例如: `https://sub.example.com/api/sing?url=订阅链接`

### 2. 在 sing-box 中导入
1. 打开 sing-box 客户端
2. 点击右上角 "+" 按钮
3. 选择 "从 URL 导入"
4. 粘贴订阅链接并确认

## 支持协议
- Shadowsocks (SS)
- VMess
- Trojan
- Hysteria2
- VLESS

## 配置说明

### DNS 设置
- 国外域名: Cloudflare (1.1.1.1)
- 国内域名: 阿里 DNS (223.5.5.5)

### 入站端口
- TUN: 系统代理
- SOCKS: 127.0.0.1:2333
- Mixed: 127.0.0.1:2334

### 分流规则
- 国内域名直连
- 国外域名代理
- 广告域名拦截
- DNS 查询分流
- 自动检测网络

### 节点分组
- Manual: 手动选择节点
  - 支持选择: Auto / DIRECT / 所有节点
  - 默认: Auto
- Auto: 自动选择延迟最低节点
  - 测试间隔: 300秒
  - 测试链接: www.gstatic.com/generate_204

### 节点命名格式
- 格式: {国旗} {地区} {序号} | {倍率}
- 示例: 🇭🇰 香港 01 | 0.5x
- 自动添加国旗和序号
- 保留原有倍率信息

## 注意事项
- 建议使用 HTTPS 链接
- 确保订阅链接可以正常访问
- 如遇导入失败，检查原始订阅是否有效
- 配置会自动更新和应用
